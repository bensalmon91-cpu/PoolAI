Finding Your Controller IP

Ezetrol Touch
  1. Press Menu on the controller
  2. Navigate to Network or Settings
  3. Look for IP Address
BAYROL
  1. Press Settings button
  2. Navigate to Network Info
  3. IP Address is displayed on screen
DULCOPOOL
  1. Go to System menu
  2. Select IP Config
  3. The IP address is shown
Walchem WebMaster
  1. Press Menu on the controller
  2. Navigate to Communications → Modbus TCP
  3. IP Address is displayed on screen

Don't know? Your pool installer should have this information.

Connecting RS485 Water Testers

  1. Hardware: Connect your RS485 adapter to the Pi's USB port
  2. Wiring: Connect A/B (or D+/D-) terminals to your water tester
  3. Serial Port: Usually /dev/ttyUSB0 for USB adapters
  4. Baud Rate: Check your device manual (common: 9600)

Display Mode:

  • Merged: Data appears within the selected pool's page
