Step 2: Enter Controller IP

The Pi needs to know your controller's IP address to connect to it. This is usually set during installation.

Finding Your Controller IP

Ezetrol Touch
  1. Press Menu on the controller touchscreen
  2. Navigate to Network or Ethernet Settings
  3. The IP address is displayed (e.g., 192.168.200.11)
BAYROL
  1. Press Settings button
  2. Navigate to Network Info
  3. IP Address is shown on screen
DULCOPOOL
  1. Go to System menu
  2. Select IP Config
  3. The IP address is displayed
Don't know? Your pool installer should have this information. The controller IP was set during installation.
