# PWA Icons for PoolAIssistant Local

This directory should contain icons for the local Pi web interface PWA support.

## Required Icons

- `icon-192.png` - 192x192 standard icon
- `icon-512.png` - 512x512 standard icon
- `apple-touch-icon.png` - 180x180 for iOS
- `favicon-32.png` - 32x32 favicon

## Notes

The local Pi interface has limited PWA support (no install prompt due to HTTP),
but these icons enable "Add to Home Screen" functionality on mobile browsers.

Use the same source icon as the cloud portal but consider using a slightly
different color scheme to distinguish local vs cloud access.
